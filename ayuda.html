Ayuda para el Plugin "User Roles and Capabilities Display"

Descripción General

Este plugin permite a los administradores de sitios de WordPress visualizar de manera sencilla todos los roles de usuario disponibles en el sitio, junto con sus capacidades activas. La información se muestra a través de un shortcode [display_user_roles], que puede ser insertado en cualquier página o entrada.

Funcionamiento

El plugin define un shortcode llamado [display_user_roles]. Cuando este shortcode se inserta en una página o entrada, el plugin genera una lista de todos los roles de usuario disponibles en el sitio de WordPress, junto con sus capacidades activas. La salida se presenta en un formato ordenado para facilitar la lectura.

Utilidad para Desarrolladores

Este plugin es especialmente útil para desarrolladores y administradores de WordPress, ya que permite:

  • Comprender rápidamente la configuración de roles y capacidades en un sitio de WordPress.
  • Diagnosticar y resolver problemas relacionados con permisos y accesos de los usuarios.
  • Ayudar en la configuración y el desarrollo de plugins y temas, asegurando que las capacidades de usuario se asignen adecuadamente.
  • Proporcionar una herramienta rápida para auditar los roles y capacidades existentes en un sitio.

Uso del Plugin

Para utilizar el plugin, simplemente inserta el shortcode [display_user_roles] en cualquier página o entrada donde desees mostrar la lista de roles y capacidades. No requiere configuración adicional.
